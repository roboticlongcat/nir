### 📥 ТЕКСТ ДЛЯ ОБРАБОТКИ (страницы 1-5):
[СТРАНИЦА 1]
Для систем радиомоторига характерна следующая задача: необходимо по радиоизлучению источника (radiolokatора, спутникового телефон и т.д.) определить его координаты. Такая задача решается, например, в системах спутниковой связи для обнаружения передатчика, неанклированного использования спутник-регранслятор [1, 2], для управления движением ракеты самонаввения [3], для определения координат судна, потерпевшего бедствие (в этом случае на судне включается специальный радиомяк), и во многих других случаях  
 
 

 В настоящее момент для решения задачи определения координат получили распространение многополизионные пассивные радиолокационные станции (MПРЛС). Определим [4] МПРЛС как радиолокационную систему, включаяную несколько разнесенных в пространстве приемных позиций, в которой получаем имя информации об объектах обработываются совместно. Владая совместной обработке информации о каждом объекте, получаем в разнесенных точках пространства, достигаются основные приемущества MПРЛС  
 
 

 Одним из возможных мест расположения регистрации аппаратуры МПРЛС является борт космического аппарата (KA). Так как по определению МПРЛС система должна составить из нескольких разнесенных приемных позиций, то необходимо использовать группировки из нескольких КА  
 
 

 Для определения координата объекта на практике применяют три основных методы: пелеггационный [5], разностно-дальномерный [4, 6], дополнительный [7]. Каждый из перечисленных методов позволяет решить задачу определения координат источника (ИРИ) при условии выполнения некоторых требований
